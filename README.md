# Sales Forecasting

## 📖 Project Overview
This project predicts future sales using **time series forecasting techniques** in Python.

## 🚀 Tech Stack
- Python
- Pandas
- Matplotlib / Seaborn
- Scikit-learn / Statsmodels

## 📊 Dataset
- Sample dataset: Monthly sales data (CSV file).

## 🔮 Output
- Forecasted sales chart
- Accuracy metrics
